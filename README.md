# Build Trust, Not Vanity — Simple GitHub Pages Repo With Chapter Visuals

This package is for a **separate GitHub repository**.

## Recommended repository name

`build-trust-not-vanity`

Live URL:

`https://ivandimitry.github.io/build-trust-not-vanity/`

## Simple upload method

1. Create a new public GitHub repo:
   `build-trust-not-vanity`

2. Upload **all files from this ZIP directly into the repo root**.
   Do not create extra folders.

3. Go to:
   `Settings → Pages`

4. Use:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Save and wait 1–3 minutes.

## What is included

- `index.html`
- the public PDF
- cover images
- social preview image
- all chapter visual PNG files
- `.nojekyll`
- `README.md`
- `FILES_TO_UPLOAD.txt`
